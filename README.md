Makes filled soul gems in nearby containers available when enchanting or recharging.

The omwaddon adds an item that when interacted with in the world will open the enchanting menu. It can be picked up by opening your inventory first and then picking it up there by clicking it.
The item id is "easyenchant_enchanting_book".

Run this command to get the item:

player->additem easyenchant_enchanting_book 1
